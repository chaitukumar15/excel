Notes — Every Topic

1. Cell References

Relative (A1) shifts when copied. Absolute ($A$1) stays fixed. Mixed ($A1 or A$1) locks only row or column. Press F4 to toggle.

2. Logical Functions

IF tests a condition; IFS handles multiple conditions without nesting.

=IFS(A1>90,"A",A1>75,"B",TRUE,"C")

AND/OR combine conditions inside IF for multi-criteria checks.

3. Lookup Functions

VLOOKUP searches the first column and returns a value from a specified column to the right — limited to left-to-right lookups.

=VLOOKUP(lookup_value, table, col_index, FALSE)

INDEX-MATCH and XLOOKUP overcome this — they search in any direction.

=XLOOKUP(lookup_value, lookup_array, return_array)

4. Text Functions

  • LEFT/RIGHT/MID extract substrings
  • TRIM removes extra spaces
  • TEXTJOIN combines text with a delimiter, optionally ignoring blanks
  • FIND (case-sensitive) / SEARCH (not) locate substrings

5. Date & Time

  • DATEDIF(start,end,"Y"/"M"/"D") — difference in years/months/days
  • EOMONTH(date,months) — last day of a month n months away
  • NETWORKDAYS(start,end) — working days excluding weekends

6. Error Handling

Wrap formulas to avoid #N/A or #DIV/0 errors showing to end users.

=IFERROR(A1/B1, "N/A")

7. Data Cleaning

  • Remove Duplicates (Data tab) deletes repeated rows based on chosen columns
  • Flash Fill auto-detects a pattern (e.g. splitting names) from one example
  • Text to Columns splits a column by delimiter or fixed width

8. Power Query Basics

Get Data → import from file/web/database. Use the Power Query Editor to remove columns, change data types, filter rows, and merge or append queries before loading back to the worksheet.

9. Conditional Aggregation

=SUMIFS(sum_range, range1, crit1, range2, crit2)

COUNTIFS and AVERAGEIFS follow the same pattern — range/criteria pairs, all conditions must be true (AND logic).

10. PivotTables

Drag fields into Rows, Columns, Values, and Filters areas to summarize large datasets instantly. Right-click → Value Field Settings to change Sum/Count/Average. Add Slicers for interactive filtering and refresh after source data changes.

11. Charts & Visualization

  • Bar/column — compare categories
  • Line — trends over time
  • Pie — proportions of a whole (best with few categories)
  • Scatter — relationship between two numeric variables
  • Sparklines — tiny in-cell trend charts

12. Dynamic Array Functions

  • UNIQUE(range) — list distinct values
  • SORT(range, col, order) — sort dynamically
  • FILTER(range, condition) — extract matching rows
  • SEQUENCE(rows, cols, start, step) — generate number series

13. What-If Analysis

  • Goal Seek — finds an input value to reach a target output
  • Data Tables — shows how output changes across a range of inputs
  • Scenario Manager — saves and compares different input sets

14. Statistical Functions

  • STDEV.S/STDEV.P — sample/population standard deviation
  • CORREL — correlation coefficient between two ranges
  • TREND / FORECAST — predict values via linear regression

15. Macros & Automation

Record Macro (Developer tab) captures a sequence of actions as VBA code to replay later. Assign to a button or shortcut key for one-click repetitive tasks like formatting or report generation.
